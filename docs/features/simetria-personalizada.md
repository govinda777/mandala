## 🚧 Tarefa: Simetria Personalizada

### 📊 Análise
- **Objetivo**: Adicionar suporte a simetria de espelhamento (estilo caleidoscópio) para criar padrões simétricos mais complexos.
- **Impacto Visual**: As pétalas serão desenhadas não apenas ao longo do círculo, mas refletidas através de múltiplos eixos, gerando um efeito visual intrigante de caleidoscópio.
- **Complexidade**: Média
- **Dependências**: Nenhuma

### 🧮 Fundamento Matemático/Científico
- **Fórmula/Algoritmo**: O ângulo refletido em relação a um eixo $\alpha$ é dado por: $\theta_{\text{refletido}} = 2\alpha - \theta$
- **Referência**: Simetria de reflexão em coordenadas polares.
- **Exemplo**: Para um ângulo de eixo $\alpha = 90^\circ$ ($\pi/2$), um ponto no ângulo $\theta = 30^\circ$ ($\pi/6$) seria refletido para $2(90^\circ) - 30^\circ = 150^\circ$ ($5\pi/6$).
